# Cadastro de Produtos

API simples de cadastro de produtos desenvolvida com Node e Express.
Guarda um array de produtos em um arquivo.

Possui as rotas:
- Cadastrar produto
- Buscar produto
- Buscar produto por id
- Deletar produto
- Alterar produto
