# Cadastro de Produtos

API simples de cadastro de produtos desenvolvida com Node e Express.
Guarda um array de produtos em um arquivo.

Possui as rotas:
- Cadastro de produto
- Busca de produto
- Busca de produto por id
- Delete produto
- Alterar produto
